# BioRAI Lab Website - NTU New Website

## Overview
This is the redesigned website for the Bio-inspired AI and Robotics (BioRAI) Lab at Nanyang Technological University (NTU), Singapore. The website has been built using the educational template and customized with content from the lab's research and activities.

## Folder Structure
- **NTU new website/** - Main website folder (renamed from "learn-educational-free-responsive-web-template")
- **assets/** - Contains CSS, JavaScript, images, and fonts
- **HTML Pages** - All main pages for the website

## Navigation Structure
The website includes the following main navigation tabs:

1. **Home** (index.html) - Landing page with overview of the lab
2. **Research** (research.html) - Research areas and focus
3. **People** (people.html) - Team members and students
4. **Publication** (publication.html) - Published research papers
5. **News** (news.html) - Latest updates and announcements
6. **Open Source** (opensource.html) - GitHub repositories and open-source projects
7. **Contact** (contact.html) - Contact information and form
